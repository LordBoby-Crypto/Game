Void Breach FPS

A lightweight first-person 3D shooter for iPhone + Backbone using the browser Gamepad API. It is raycast 3D, so it runs as a single HTML file on GitHub Pages.

Left stick: move / strafe
Right stick: look around
R2 / right trigger: shoot
Square / X button: reload
Touch fallback: drag left side to move, right side to look, tap to shoot

Press a Backbone button after the page loads so Safari exposes the controller.
